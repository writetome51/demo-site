Steve Thompson Web App Demos

  • Table of Users (with expanding rows)

    A simple data display demo, built in Angular 7.

    Queries third-party API and allows you to search the results.

    Clicking on a user reveals more info about that user.

    Github repository

  • Panterest

    A recipe search site, built in Angular 5.

    Queries third-party API to perform searches.

    Allows you to log in with a Google account, comment on recipes, and save favorites.

    Uses Firebase to store user data.

    Github repository
